SMS Compliance
// Bruhn Freeman · Text Messaging Program

This page describes how Bruhn Freeman collects consent, delivers, and manages SMS text messages sent to users of the PhoneTester service at bruhnfreeman.com. All SMS activity is conducted in full compliance with the TCPA, CTIA guidelines, and carrier A2P 10DLC requirements. No mobile opt-in data is ever shared with third parties, affiliates, or lead generators for marketing or promotional purposes at any time.

📋
Program Name
Bruhn Freeman — PhoneTester
SMS notifications are sent exclusively in connection with registered user accounts on the PhoneTester voice quality analysis platform at bruhnfreeman.com.
📨
Message Types
Messages fall into three categories:
• MFA: Security codes for login verification
• Announcements: Feature updates and service changes
• Diagnostics: PhoneTester tool notifications
Each category requires separate opt-in consent.
🔢
Message Frequency
Message frequency varies based on account activity. Typically 1–3 messages are sent per registration or test session. Users will not receive unsolicited recurring messages.
💰
Rates & Carriers
Message and data rates may apply depending on your mobile carrier plan. Bruhn Freeman does not charge for SMS messages. Carrier charges are the responsibility of the recipient.
// Section 01

How We Obtain Consent (Opt-In)

Bruhn Freeman uses a web-based opt-in method. Consent is collected separately from account registration through the user's profile settings page at bruhnfreeman.com/profile. SMS opt-in is never required as a condition of creating an account or using our services.

💬

Users can enable SMS notifications for three distinct purposes:

1. Multi-Factor Authentication (MFA) — Security codes for account login
2. Site Announcements — Updates about new features and service changes
3. Testing & Diagnostic Tools — Messages from tools like PhoneTester

Each option has a clear description of message content and approximate frequency. All toggles are off by default. Links to our Privacy Policy and this SMS Compliance page are displayed within the SMS settings section.

  • All SMS toggles are disabled by default — users must explicitly enable each one.
  • Opt-in is completely voluntary. Users may use all site features without enabling SMS.
  • Consent is not a condition of any purchase or access to services.
  • The opt-in URL is: https://www.bruhnfreeman.com/profile (SMS Notifications section).
  • Consent records including timestamp and IP address are stored securely for each SMS category.
  • Users may enable or disable any SMS category independently at any time.
  • No mobile opt-in data will be shared with or sold to third parties, affiliates, or lead generators for marketing or promotional purposes at any time.
// Section 02

How to Opt Out (STOP)

Users may opt out of SMS messages at any time using any of the following methods:

STOP CANCEL QUIT UNSUBSCRIBE END
  • Reply any of the keywords above to any message received from our number to immediately opt out.
  • Upon receiving a STOP request, a single confirmation message will be sent confirming the opt-out. No further messages will be sent.
  • Users may also opt out by contacting us directly at support@bruhnfreeman.com.
  • Opt-out requests are processed immediately and are permanent unless the user explicitly re-opts in.
// Section 03

Help & Support Keywords

Users may reply to any message with the following keywords to receive assistance:

HELP INFO

Replying HELP returns a message containing the program name, brief description, opt-out instructions, and our support contact. Replying INFO returns general account information.

📟

Sample HELP response: "Bruhn Freeman PhoneTester: SMS alerts for your account. Msg&data rates may apply. Reply STOP to cancel. Support: support@bruhnfreeman.com or bruhnfreeman.com/#contact"

// Section 04

Data Privacy & Non-Sharing Commitment

  • No mobile opt-in data will be shared with third parties, affiliates, or lead generators for marketing or promotional purposes at any time.
  • Phone numbers are collected solely to deliver account-related SMS notifications.
  • We do not sell, rent, or trade mobile phone numbers to any outside party.
  • SMS data is processed through Twilio solely as a delivery mechanism. Twilio does not receive marketing rights to user data.
  • All data handling is governed by our full Privacy Policy.
  • Users may request deletion of their data at any time by contacting support@bruhnfreeman.com.
// Section 05

Sample Message Content

The following are representative examples of messages sent by Bruhn Freeman:

  • Verification: "Your Bruhn Freeman account has been verified. You may now log in at bruhnfreeman.com. Reply STOP to opt out."
  • Test Code Delivery: "Your PhoneTester code words are ready. Check your email for the 5-word sequence and call number. Reply STOP to opt out."
  • Security Alert: "New login detected on your Bruhn Freeman account. If this wasn't you, contact support@bruhnfreeman.com immediately. Reply STOP to opt out."
// Section 06

Sender Information

  • Messages are sent from a dedicated Twilio-provisioned long code (10DLC) number registered to Bruhn Freeman.
  • The sending number is consistent across all messages so users can identify our program.
  • We do not use shared short codes or shared sender pools.
  • Our number is registered under the A2P 10DLC framework with all major US carriers.
